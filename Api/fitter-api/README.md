# FITTER API

The Fitter API provides a RESTful API for the Fitter application

## Getting Started

To get started with the Fitter API, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Siyabonga003/fitter-api.git
    ```

2. Navigate to the project directory:
    ```bash
    cd fitter-api
    ```

3. Build the project:
    ```bash
    ./gradlew build
    ```

4. Run the application:
    ```bash
    ./gradlew bootRun
    ```

Swagger UI: http://localhost:8080/swagger-ui/index.html
